# 🍽️ Meal Explorer

A modern and responsive meal discovery web application built with **HTML**, **CSS**, **Bootstrap**, **JavaScript**, and **TheMealDB API**. Browse meals by category, area, or ingredients, search for your favorite recipes, and explore detailed cooking instructions in a clean and user-friendly interface.

## ✨ Features

* 🍽 Browse all available meals
* 🔍 Search meals by name
* 🔤 Search meals by first letter
* 📂 Browse meals by category
* 🌍 Explore meals by area (country)
* 🥩 Browse meals by main ingredient
* 📖 View complete meal details
* 🥗 Display recipe ingredients
* 📝 Cooking instructions
* ▶️ Watch recipe on YouTube
* 🌐 Visit the original recipe source
* 📱 Fully responsive design
* ⚡ Fast and dynamic UI using API data

## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/MostafaMohamid/ingredientsMeals.git
```

Open the project folder

```bash
cd ingredientsMeals
```

Run the application by opening:

```text
index.html
```

No installation or build process is required.
